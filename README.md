# oneaura's CPS Counter

CPS Counter for the latest versions of Minecraft with very customizable features!

## Features

| Feature | Description |
| :--- | :--- |
| Master Switch | Completely toggle on/off the mod! |
| Draggable HUD | You can put the mod anywhere on the HUD! |
| Custom Text Color | Changeable text color. **(hex color code)** |
| "CPS" Label | You can either change the "CPS" label or completely disable it! |
| Text Style | The style of the text can be changed to normal, bold, and italic. |
| Background | You can change the radius and the color of the background or completely disable the background! |
